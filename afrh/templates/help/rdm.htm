Using the RDM (Reference Data Manager)
Introduction

The RDM is a robust way of handling all of the behind-the-scenes concepts used in the AFRH-W database. While creating new resources, or editing existing resources, you will notice that most data entry is done by selecting from predetermined dropdown lists, also known as controlled vocabularies. Controlled vocabularies ensure clean data, and the RDM is where you manage them.

In the panel on the left, you can see the following four menus:

  • The Arches menu holds all of the concept schemes in the database
  • The Dropdown Lists menu holds the controlled vocabularies that are passed to the data input forms
  • The Entity Nodes menu lists all of the entity nodes (types of information collected) present in the database
  • The Candidates menu is used for more advanced procedures, and is not necessary for the actions described below
Adding a New Concept

Undoubtedly, new concepts will need to be added to the database in the future. The following steps will explain how to add Staircase as a new Component Type concept.

  1. Open the Arches menu, scroll down and select the Component Type concept scheme. In the right panel, you can see all of the existing concepts.
  2. Click the Manage menu, and select Add Child.
  3. Type Staircase into the Label box. You can add a note to this concept if you like; it will be visible to anyone who views the concept in the RDM later.
  4. Click Save Changes. You have now added a new Component Type, and you'll see Staircase in the list of Keyword concepts.

Now that you have added the new concept in the Arches concept scheme, you need to add it to the appropriate dropdown menu.

  1. Open the Dropdown Lists menu, and scroll down to the Component_Type.E55 dropdown. In the right panel, you can see the current contents of the dropdown.
  2. Click on Add dropdown entry, and begin to type Staircase. The new keyword will appear in the search bar, now that it has been added to the Arches concept scheme.
  3. Select the concept, and click Save.

The new concept Staircase will now be available to any Component Type dropdown.

Adding an Alternate Label

In some cases, it may be desirable to use two completely different terms for the same concept. For example, Chimney and Smokestack. That way, searches or entries using either name will be treated the same. The following steps will show how to add Smokestack as an alternate label for the concept Chimney.

  1. Open the Arches menu, scroll down and select the Component Type concept scheme.
  2. Click on Chimney.
  3. Click Add label, and the Add Concept Label window will open.
  4. Type in Smokestack, and indicate that this is an altLabel ("alternate label"). Click Save.

Now, if a resource is entered that includes Chimney as a component, a search for Smokestack will also find it.
