Using the Reference Data Manager (RDM)
Introduction

The Reference Data Manager (RDM) is a robust way of handling all of the behind-the-scenes concepts used in the CRHIM database. While creating new resources, or editing existing resources, you will notice that most data entry is done by selecting from predetermined dropdown lists, also known as controlled vocabularies. Controlled vocabularies ensure clean data, and the RDM is where you manage them.

In the panel on the left, you can see the following four menus:

  • The Arches menu holds all of the concept schemes in the database
  • The Dropdown Lists menu holds the controlled vocabularies that are passed to the data input forms
  • The Entity Nodes menu lists all of the entity nodes (types of information collected) present in the database
  • The Candidates menu is used for more advanced procedures, and is not necessary for the actions described below
Adding a New Concept

Undoubtedly, new concepts will need to be added to the CRHIM in the future. For example, you may find the need to add a new Keyword concept, so it can be associated with a new resource. The following steps will explain how to do so.

  1. Open the Arches menu, scroll down and select the Keyword concept scheme. In the right panel, you can see all of the existing concepts in the Keyword concept scheme.
  2. Click the Manage menu, and select Add Child.
  3. Create a new Keyword ("Logging", for example) by entering it in the Label box. You can add a note to this concept if you like; it will be visible to anyone who views the concept in the RDM later.
  4. Click Save Changes. You have now added a new Keyword, and you'll see "Logging" in the list of Keyword concepts.
  5. Too add this new Keyword concept to the Keywords dropdown list (so it can be used in the creation of new resources), close the Arches menu, and open the Dropdown Lists menu.
  6. Scroll down to the Keyword.E55 entry, and open that collection. You'll see the list of concepts that already exist in the dropdown.
  7. Click on Add dropdown entry, and begin to type "Logging". The new keyword will appear in the search bar, now that it has been added to the database (steps 1-4).
  8. Select the concept, and click Save.

The new concept "Logging" will now be available to any Keyword dropdowns.

Adding an Alternate Label

In some cases, it may be desirable to use two completely different terms for a single concept, for example "post-in-ground" (English) and "poteaux-en-terre" (French). To accommodate this, we can use the RDM to add extra labels to an existing concept. That way, searches or entries using either name will be treated the same. The following steps will show how to add "poteaux-en-terre" as an alternate label for the concept "post-in-ground".

  1. Open the Arches menu, scroll down and select the Construction Technique concept scheme.
  2. Because "post-in-ground" has been classified as a subset of the more general "framing" category, click Framing.
  3. Now you can see the concept you're looking for, click on post-in-ground.
  4. Click Add label, and the Add Concept Label window will open.
  5. Simply type in poteaux-en-terre, and be sure to indicate that this is an altLabel ("alternate label"). Click Save.

Now, if a resource is entered that includes "post-in-ground" as a construction technique, a search for "poteaux-en-terre" will match it.
